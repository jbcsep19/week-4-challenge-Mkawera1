About the Java Bootcamp

The Java Web Developer Boot Camp is an 8 hour a day (9 am - 5 pm, M-F) 8-week, immersive software engineering program funded by the Department of Labor. This course aims to increase the number of skilled software developers in this country. It accepts candidates who cannot afford the cost of a coding boot camp. The fees will are covered by the program.

The Bootcamp takes you from your first java program to developing a web application. This course gives you a fast start on your career so you can take it to the next level. The course teaches you by forcing you to write over 130 java applications, including 5 major web applications. Some of the modules covered include GitHub,Program Design & Algorithms,Core Java,Object-oriented Java and last but not list HTML/CSS/Bootstrap The Programming challenges so far, have kept us on our toes.

The ideal candidate is an Information Systems or Computer Science major who is either at least a rising junior or graduated within the last six years. If a student doesn't meet those criteria, they can participate in a special one week mini-camp to assess their ability to learn in this environment. Applicants also must be unemployed or underemployed to qualify for the grant. In the state of Maryland, a person who makes $42,000 or less is underemployed. Must be able to work in the United States. Students must apply to be considered for the program.
